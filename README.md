# beautifulbest

高端定制服饰协同助手。

这是一个占位 README，用于初始化仓库并启用 GitHub Pages。访问 Pages 站点（如已启用）通常位于：

https://jhonyidao-hub.github.io/beautifulbest/

部署说明：
1. 在本地克隆仓库并添加代码：
   - git clone https://github.com/jhonyidao-hub/beautifulbest.git
   - cd beautifulbest
   - 编辑或添加文件
   - git add .
   - git commit -m "feat: add project files"
   - git push origin main

2. 如果使用 GitHub Pages，请在 Settings → Pages 中确认发布分支为 main，并确保仓库根目录包含 index.html 或构建输出.
